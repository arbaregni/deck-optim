## How?

### How to run:

1. Install cargo.
2. Build this project `cargo build`.
3. Run this with cargo or via the stand alone exectuble. `cargo run -- -d <your deck>.json`.

### How to test

1. `cargo test`
2. To check coverage: `cargo llvm-cov --open`.
